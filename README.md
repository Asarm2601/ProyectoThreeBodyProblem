# Proyecto ABET : Simulación del problema de los tres cuerpos
Para una correcta ejecución del programa, se debe ejecutar el archivo .cpp del método numérico a utilizar y luego seleccionar el sistema de preferencia(se mostrará un menú de opciones).
Luego, se debe ejecutar el archivo Graficas.ipynb de la carpeta /Codigo/Python para visualizar la representación gráfica de los datos obtenidos por la simulación.
